# 1. a 标签的用法：

作用超链接网址
<a href = "网址" target = "black"名称/a>

# 2. img 标签的用法

发出 get 请求 展示一张图片

<img src="图片" alt="如果加载图片失败，显示a以后用alt属性提醒用户" >

# 3. table 标签的用法

创建一个容器 所有表格内容都在里面

<table>
  <thead>... ...</thead>
  <tbody>... ...</tbody>
  <tfoot>... ...</tfoot>
</table>
